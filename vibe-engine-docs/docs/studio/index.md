# VibeEngine Studio

The **VibeEngine Studio** is the companion IDE for the VibeEngine. It provides a real-time, high-fidelity environment for building 3D worlds without writing extensive code.

## Key Features

- **Quantum Fluid UI**: A lag-free editor layout with real-time feedback.
- **Hierarchy Panel**: Manage all entities and their organizational structure.
- **Inspector Panel**: Detailed control over component properties and data.
- **Scene Viewport**: High-performance 3D rendering with built-in gizmos.
- **Neural Copilot**: Integrated AI to help you build and debug.

## Workspace Layout

The editor is divided into several main areas:

- **Top Bar**: Scene controls (Play/Pause/Stop), project settings, and layout toggles.
- **Left Sidebar**: The **Hierarchy Panel**, which shows the scene tree.
- **Right Sidebar**: The **Inspector Panel**, showing components of the selected entity.
- **Center**: The **3D Viewport**, where you can visualize and interact with the scene.
- **Bottom**: The **Asset Manager**, used to import models, textures, and scripts.

## Navigation

- **Rotate**: `Right Click` + Drag
- **Pan**: `Middle Click` + Drag
- **Zoom**: `Scroll Wheel`
- **Focus**: Press `F` while an entity is selected.
